## Binary Sunset FE Challenge – AG Grid Data Table

### Objective

Develop a **high-performance, interactive data table** using **AG Grid** that demonstrates:
- **Advanced cell rendering**
- **Dynamic, row-level calculations**
- **Real-time, input-driven updates**

### Quickstart guide

This app is running on React and Vite. 

To install all the dependeincies run:

`npm install`

To run the app with 10k data set use:

`npm run dev`

## Testing

To start the unit tests use:

`npm run test`

To start the E2E tests (cypress) use:

`npm run dev` for starting up your dev envoriment and then `npx cypess run` for running e2e test suite
